## 低代码
### 本质 —— 降本增效
1. 降低开发成本
=> a. 强制统一组件库复用 b. 提升系统一致性 c. 降低开发资源的投入
=>
a. 物料堆的建立（直联、整合）
型态：
el-form el-form-item
el-input
zw-form-input
特点：
业务解耦、无感知的热装载

b. 系统一致性规范（交互、UI、逻辑） —— 原子设计

c. 开发投入流程

2. 传统型配置平台
Dreamweaver、xcode、IDE —— 面向骨架样式型配置工具
=> no-code，但更专注于布局

3. no-code & low-code —— 混合、low-code、no-code
no-code: 富布局、轻逻辑的页面
low-code: 重逻辑 —— 配置为主、代码做逻辑补充
